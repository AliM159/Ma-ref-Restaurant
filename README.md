# Ma-ref Restaurant Website

This is a simple static website for a restaurant, built using **HTML** and **CSS**.  
It focuses on clear layout, visual presentation of dishes, and clean navigation between pages.

## Features

- Multi-page layout (e.g., home, menu, gallery, etc.)
- Static content created with semantic HTML
- Consistent styling using a custom CSS stylesheet
- Image sections to showcase food and atmosphere
- Simple, easy-to-read typography and colour scheme

> Note: This project is **purely front-end**. There is **no JavaScript** and no functional contact form or backend.

## Technologies Used

- **HTML5**
- **CSS3**

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Browse through the pages using the navigation links.

No build tools or servers are required — this is a static site.
